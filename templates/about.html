Colorado Activist feed aggregates social media posts of organizations and individuals that are working for progressive goals.

If you have questions, an issue, would like to be removed, or would like to be added to the feed, contact us by opening a github issue here, or by sending an email to colorado.activism@gmail.com.

Full source code is available at github.com/ColoradoActivistFeed/RSSAggregator. RSS Aggregator is released under GPL-3.0.
